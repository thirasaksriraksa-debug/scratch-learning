หน่วยที่ 1 รู้จัก Scratch

1.1 ความหมายของ Scratch

Scratch เป็นโปรแกรมสำหรับการเรียนรู้การเขียนโปรแกรม โดยใช้การลากบล็อกคำสั่งมาประกอบกัน ช่วยให้ผู้เรียนสามารถสร้างเกม แอนิเมชัน และสื่อการเรียนรู้ได้ง่าย

1.2 ประโยชน์ของ Scratch

  • ช่วยฝึกการคิดอย่างเป็นระบบ
  • ช่วยพัฒนาทักษะการแก้ปัญหา
  • สามารถสร้างเกมและแอนิเมชันได้
  • ช่วยให้เข้าใจพื้นฐานการเขียนโปรแกรม

1.3 ส่วนประกอบของโปรแกรม

  • Stage พื้นที่แสดงผล
  • Sprite ตัวละคร
  • Blocks Palette คลังบล็อกคำสั่ง
  • Scripts Area พื้นที่เขียนโปรแกรม

1.4 ประเภทของบล็อกคำสั่ง

  • Motion
  • Looks
  • Sound
  • Events
  • Control

1.5 การสร้างตัวละครและฉาก

ผู้ใช้สามารถสร้างตัวละคร (Sprite) และฉาก (Stage) ได้จากคลังของโปรแกรม หรืออัปโหลดรูปภาพของตนเอง

บทเรียน Scratch
